# My First Web Art Template

This is my first web art template.

It uses simple HTML, JavaScript, and CSS to create an animated web page with falling blossoms.

## 🌸 What is this?

A basic interactive web page using:
- **HTML** (`index.html`)
- **JavaScript** (in a `<script>` tag in the HTML file)
- **CSS** (inline in the HTML `<style>` block)



## 📁 Files

- `index.html` — The main page, including all HTML, CSS, and JavaScript
- `imgs/sakura.png` — Blossom image used in the animation

## 🖼️ How to Use

- Open `index.html` in your browser, or visit the GitHub Pages link above.
- You can copy or edit the files to make your own web art.

---

ALWAYS USE A PNG OR SVG IMG!
